Lexata

Lead a group of 6 and worked on the backend using Python Flask to build a web application for a startup company, Lexata.

The purpose of the web app was to simplify the search for corporate legal questions using machine learning. Used the OpenAI API, along with its davinci text-model generation endpoint to answers questions queries made by the user.

Users can create accounts and make searches that are relevant to corporate law, and the answers will be given based on the Delware Code, which was the target legal code undertaken for this project. Below you will find a picture with the result given a sample query. There you will find a QnA type of answer from the GPT3-text-completion endpoint, and below that, the n most relevant sections of the legal code, based on cosine similarity of vectorized text, calculated and stored by Pinecone API.
